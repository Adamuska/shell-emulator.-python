# Shell Emulator

## Описание

**Shell Emulator** — это эмулятор командной оболочки на Python, который имитирует файловую систему, загруженную из ZIP-архива. Пользователи могут перемещаться по виртуальной файловой системе, выполнять базовые команды, такие как `ls`, `cd`, и получать информацию о текущем пользователе. Оболочка работает с виртуальной файловой системой, которая загружается из указанного ZIP-файла.

## Основные команды

- `ls` — выводит список файлов и папок в текущей директории.
- `cd <directory>` — изменяет текущую директорию. Поддерживаются относительные пути, такие как `..` для возврата на один уровень вверх.
- `who` — выводит информацию о текущем пользователе.
- `rev` — выводит текущую директорию в обратном порядке.
- `exit` — завершает работу эмулятора.

## Установка и запуск

1. Клонируйте репозиторий или скопируйте код к себе на локальный компьютер.
2. Подготовьте ZIP-архив с файловой системой (например, `virtual_fs.zip`), которая будет использоваться в эмуляторе.
3. Убедитесь, что у вас установлен Python версии 3.x.

### Пример использования:

Запустите эмулятор командной оболочки, указав хостнейм и путь к ZIP-файлу:

```bash
python shell_emulator.py <hostname> <zip_path>
```
